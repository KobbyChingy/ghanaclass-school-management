# GhanaClass School Management - Features Documentation

This companion copy mirrors the current supported portal set.

## Supported portals

- Admin
- Director
- Headmaster
- Headmistress
- Teacher
- Accountant
- Shop

## Retired portals

These portals are no longer part of the supported navigation or router configuration:

- Parent
- Deputy Head
- Secretary
- Security
- Library
- Infirmary
- Chef
- ICT Lab
- Science Lab

## Guardian data note

Guardian and parent account records remain in the data model for student linkage and staff-to-guardian communication. What was removed is the dedicated Parent portal and its route tree.
